# 🌦️ WeatherIQ Analytics Dashboard

Real-time weather intelligence dashboard built using Streamlit and OpenWeather API.

---

## 🔹 Features
- 🌍 Multi-city weather monitoring  
- 🌡️ Temperature (°C / °F toggle)  
- 🌫️ Air Quality Index (AQI)  
- ☀️ UV Index with risk levels  
- 🌬️ Wind speed and direction  
- 📈 Temperature trend visualization  
- ⚠️ Smart weather alerts  

---

## 🔹 Cities Covered
Chennai, Coimbatore, Mumbai, Delhi, Salem, Bangalore, Hyderabad

---

## 🔹 Tech Stack
- Python  
- Streamlit  
- OpenWeather API  

---

## 🔹 How to Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
